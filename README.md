# 保洁调度系统

保洁公司调度保洁员的 Windows 桌面应用程序。

## 功能

- 📊 **仪表盘** - 今日任务概览，快速操作
- 📋 **任务管理** - 新建/编辑/删除任务，按状态和日期筛选
- 👥 **保洁员** - 管理保洁员信息，设置在职/离职状态
- 📅 **排班视图** - 按日期查看每个保洁员的任务安排

## 下载

前往 [Releases](https://github.com/aerofish/cleaning-scheduler/releases) 页面下载最新版本。

## 本地开发

```bash
# 安装依赖
npm install

# 运行开发版本
npm start

# 打包 Windows 版本
npm run build
```

## 技术栈

- Electron
- HTML/CSS/JavaScript
- localStorage 本地存储
