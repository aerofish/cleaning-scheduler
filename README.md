# 保洁调度系统

一个基于 Electron 的保洁员接单调度桌面应用。

## 功能

- 👨‍🔧 保洁员管理：添加、删除保洁员，记录技能和联系方式
- 📋 订单管理：创建服务订单，跟踪状态
- 🎯 订单分配：将订单分配给合适的保洁员
- 📊 仪表盘：实时查看业务统计

## 下载

前往 [Releases](https://github.com/aerofish/cleaning-scheduler/releases) 页面下载最新版本。

## 本地开发

```bash
# 安装依赖
npm install

# 运行开发版本
npm start

# 构建 Windows 可执行文件
npm run build
```

## 技术栈

- Electron
- Node.js
- 原生 HTML/CSS/JavaScript
